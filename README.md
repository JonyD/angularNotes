# angularNotes

### Pretty HTML report for jasmine test in Karma

in `package.json` add to `devDependencies`:

* karma-jasmine-html-reporter (RG)
`"karma-jasmine-html-reporter": "~0.1"`

* karma-html-file-reporter (RGY)
`"karma-htmlfile-reporter": "~0.2"`
